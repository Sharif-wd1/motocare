# HERO.IO

A modern and responsive React web application showcasing a collection of applications built by our team. Users can explore, search, and install apps, while enjoying a smooth and fast navigation experience.

---
 Live Link: https://lambent-gaufre-7db9ab.netlify.app/


## Features

- **Smooth Navigation** using React Router v7  
- **Real-time App Search** with instant filtering  
- **LocalStorage-based Installation System**  
- **Fully Responsive UI** built with Tailwind CSS  
- **Loading Animations** during route transitions  
- **Reusable Components** for clean architecture  

---

## Technologies Used
-----------------------------------------------------------------
| Category               | Technologies                          |
|------------------------|---------------------------------------|

| **Styling**            | Tailwind CSS, DaisyUI                 |
