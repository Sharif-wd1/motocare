#  MotoCare – Smart Bike Care & Maintenance Platform

MotoCare is a modern, responsive web application that provides professional bike care, maintenance, and service booking solutions.  
It is designed with a clean UI, secure authentication, and smooth animations for a premium user experience.

---

##  Live Demo
 **Live URL:** (https://motocare-sar.netlify.app/)  

---

##  Project Purpose
MotoCare helps bike owners easily find, book, and manage bike maintenance services online.  
It ensures a fast, secure, and seamless digital experience — from browsing services to managing user profiles.

---

##  Key Features

 **User Authentication**
- Secure Firebase Authentication (Sign Up / Sign In / Sign Out)
- Protected Routes for authenticated users only  
- Auto-redirect to Sign In when unauthorized

 **Dynamic Services**
- All services are dynamically loaded and displayed using reusable `ServiceCard` components  
- Professional grid layout with hover & scroll animations  

 **Profile Management**
- Users can update their display name and profile picture in real time  
- Smooth Framer Motion animations with modern UI feedback

 **Responsive Design**
- Fully responsive across mobile, tablet, and desktop  
- TailwindCSS for consistent design system

 **Error Handling**
- Custom 404 Error Page with navigation support  

 **Toast Notifications**
- Success and error alerts for all user actions using `react-toastify`

 **Modern Animations**
- Smooth UI transitions using **Framer Motion**
